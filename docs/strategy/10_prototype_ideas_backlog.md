## Backlog (9 идей)

### Auto Timesheet (R1.5)
AI генерирует табель из сессий фокуса. Проект, тег, длительность, экспорт в Toggl/Harvest.
**История:** "Фокус = автоматический учёт работы без ввода руками"
**Когда актуален:** Когда App Tracking (1.4) и Project Tags (1.8) в продакшне.

---

### Family Status Page (R1)
`busy.app/status/pavel` — что видит семья. Без приложения. Простая страница с LED-статусом в браузере + кнопка "можно зайти".
**История:** "Физический сигнал выходит за пределы комнаты без установки ничего"
**Когда актуален:** Family URL (1.11) — одна из первых фич Ring 1.

---

### Active Focus Session — Fullscreen (R0)
Полноэкранный вид во время помодоро: LED цвет, таймер, текущий проект, контекст "не беспокоить". macOS screensaver / wallpaper режим.
**История:** "Ядро продукта — момент когда человек начинает работать"
**Когда актуален:** Хорош как первый экран для pitch, самый понятный.

---

### App Library (R2)
Каталог интеграций: поиск, категории (Calendar / Smart Home / AI / Team), 1-click install, рейтинги.
**История:** "BUSY — платформа, не гаджет"
**Когда актуален:** Phase 2.0, после Developer Portal (2.3).

---

### Interruption Cost Report (R1.5)
"23 прерывания на этой неделе = -3.2 часа deep work = ~$480 при твоей ставке". Визуализация потерь с breakdown по типам прерываний.
**История:** "Цена рассеянности измерима и конкретна"
**Когда актуален:** После Interruption Cost (1.10) и App Tracking (1.4).

---

### Smart Home Scene (R2)
Focus Mode активирован → лампы тускнеют (Philips Hue), табличка на двери загорается, кофемашина запускается. Визуализация Matter automation chain.
**История:** "Физический мир реагирует на твой статус"
**Когда актуален:** Home Assistant (2.6) + HomeKit/Matter (2.7).

---

### LED Matrix Designer (R0)
Пиксель-редактор для создания кастомных анимаций LED матрицы под разные статусы. 72×16px grid, цветовые пресеты, preview.
**История:** "Личная кастомизация hardware — устройство становится твоим"
**Когда актуален:** Themes & UI (0.15) — можно делать уже сейчас.

---

### Developer Marketplace (R3)
Разработчик публикует интеграцию, видит статистику загрузок и доход (revenue share). Дашборд с earnings, reviews, версиями.
**История:** "BUSY = экономика для разработчиков, не просто API"
**Когда актуален:** Marketplace (3.4) — Ring 3, далекое будущее.

---

### Weekly Focus Report (R1.5)
Красивый digest: лучший день недели, худший час, сравнение с прошлой неделей, один совет от AI на следующую неделю. Email / PDF формат.
**История:** "BUSY сам объясняет тебе твою неделю — без ввода данных"
**Когда актуален:** После Focus Score (1.6) и Focus Heatmap.
